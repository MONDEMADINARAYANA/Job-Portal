# Job Portal

A simple Job Portal web application built using Django. Users can view available jobs with company details, location, salary, and description.

## Features
- View job listings
- Company details
- Job location
- Salary information
- Job description
- Admin panel to manage jobs

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## Installation

1. Clone the repository:
git clone https://github.com/yourusername/jobportal.git

2. Navigate to project folder:
cd jobportal

3. Install dependencies:
pip install django

4. Run migrations:
python manage.py makemigrations
python manage.py migrate

5. Start server:
python manage.py runserver

6. Open browser:
http://127.0.0.1:8000/
